## Overview

- Rails 4.0.0rc1
- Ruby 2.0.0-p0
- Backbone.js
- Dust.js (Templating)

## Installation

1. Clone the repo: `git clone git@github.com:joelbeukelman/photeasy.git`
2. Install dependencies: `bundle install`
3. Migrate the database: `rake db:migrate`
